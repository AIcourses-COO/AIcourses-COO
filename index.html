FAQ

Հաճախ տրվող հարցեր

Ո՞ր պրոֆիլով պետք է մուտք գործեմ Microsoft Learn։

Օգտագործեք ձեր հիմնական աշխատանքային Microsoft account-ը, որպեսզի collection-ի առաջընթացը և badge-երը պահպանվեն նույն պրոֆիլում։

Արդյո՞ք Microsoft assessment-ը պարտադիր է։

Այո։ Յուրաքանչյուր համապատասխան unit-ից հետո ավարտեք assessment-ը, որպեսզի կարողանաք ստանալ badge և հաստատել դասընթացի ավարտը։

Ի՞նչ պետք է անեմ Anthropic course-ից հետո։

Անցեք course-ի ներքևում ներկայացված certification-ը, ներբեռնեք կամ պահպանեք certificate-ը և ուղարկեք ձեր անմիջական ղեկավարին։

Կարո՞ղ եմ այս կայքում պահել այլ կայքերի գաղտնաբառերը։

Ոչ։ GitHub Pages-ը static և հրապարակային հարթակ է։ Գաղտնաբառերը պետք է պահվեն միայն հաստատված password manager-ում կամ սահմանափակ հասանելիությամբ ներքին համակարգում։
